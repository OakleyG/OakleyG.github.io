Foundations — Cosets, Lagrange & Cyclic Groups

Everything Builds on This

Cosets

DEFINITION · Left and Right Cosets

Let H ≤ G and a ∈ G. The left coset of H containing a is aH = {ah : h ∈ H}.

The right coset is Ha = {ha : h ∈ H}.

Key facts:

1. Cosets are not subgroups (unless a ∈ H).

2. aH = bH if and only if a⁻¹b ∈ H (cosets are equal or completely disjoint).

3. The distinct left cosets of H partition G into equal-sized pieces.

4. |aH| = |H| for all a ∈ G — the map h ↦ ah is a bijection H → aH.

Coset membership test: To check if x ∈ aH, compute a⁻¹x. If a⁻¹x ∈ H, then x ∈ aH. To check if aH = bH, compute b⁻¹a — if it's in H, the cosets are equal.

Lagrange's Theorem

★ LAGRANGE'S THEOREM — The Single Most Used Result in the Course

If G is a finite group and H ≤ G, then |H| divides |G|.

Equivalently: |G| = |H| · [G : H] where [G:H] is the number of distinct cosets of H in G.

Proof Sketch — Lagrange's Theorem

1.Partition: The distinct left cosets of H in G partition G — every element belongs to exactly one coset. (Any two cosets are either equal or disjoint.)
2.Equal sizes: f: H → aH by f(h) = ah is a bijection, so |aH| = |H| for every coset.
3.Count: G is partitioned into [G:H] cosets each of size |H|, so |G| = [G:H]·|H|, meaning |H| divides |G|. □
COROLLARIES OF LAGRANGE — Know All Four

1. Order of elements: For any a ∈ G, ord(a) divides |G|. Therefore a^|G| = e for all a ∈ G.

2. Prime order → cyclic: If |G| = p (prime), then G ≅ Cₚ. Every non-identity element has order p and generates G.

3. Index-2 ⟹ normal: If [G:H] = 2, then H ◁ G. (Only two cosets — H itself and everything else — so left and right cosets agree.)

4. Converse is FALSE: A group need not have a subgroup of every order dividing |G|. Counterexample: A₄ (order 12) has no subgroup of order 6.

The most common exam trap: "Since k | |G|, G must have a subgroup of order k." FALSE in general. Lagrange is one-directional. Cauchy covers primes; Sylow covers prime powers. For arbitrary divisors — no guarantee.

Cyclic Groups

KEY FACTS — Cyclic Groups & Subgroup Structure

G is cyclic if G = ⟨a⟩ = {aⁿ : n ∈ ℤ} for some a ∈ G.

1. Every subgroup of a cyclic group is cyclic.

2. If |G| = n, then G has exactly one subgroup of order d for each d | n.

3. In Cₙ = ⟨a⟩: the subgroup ⟨aᵏ⟩ = ⟨a^{gcd(k,n)}⟩ and has order n/gcd(k,n).

4. The number of generators of Cₙ is φ(n) (Euler's totient).

5. Cₘ × Cₙ ≅ Cₘₙ iff gcd(m,n) = 1.

Subgroups of Cₙ: One subgroup for each divisor of n. The subgroup of order k is ⟨a^{n/k}⟩. The subgroup lattice of Cₙ mirrors the divisibility lattice of n — draw it! This is how you quickly see all subgroups.

Group Actions — The Engine Behind Sylow

DEFINITION · Group Action

A group action of G on a set S is a map G × S → S, written (g, s) ↦ g·s, satisfying:

1. Identity: e·s = s for all s ∈ S.

2. Compatibility: (gh)·s = g·(h·s) for all g,h ∈ G, s ∈ S.

Orbit of s: orb(s) = {g·s : g ∈ G} — all places s can move to.

Stabilizer of s: stab(s) = {g ∈ G : g·s = s} — elements that fix s. Always a subgroup of G.

Orbit-Stabilizer Theorem: |G| = |orb(s)| · |stab(s)|

Key Actions to Know

  • Conjugation on elements: g·x = gxg⁻¹. Orbits = conjugacy classes. Stabilizer of x = Cₓ (centralizer).
  • Left mult. on cosets: g·(aH) = gaH. Used in Sylow 2 proof. Stabilizer of gH = gHg⁻¹ ∩ K (for K acting).
  • Conjugation on subgroups: g·P = gPg⁻¹. Orbits = conjugacy classes of subgroups. Stabilizer of P = N_G(P). Used in Sylow 3.
  • Cayley's Theorem: G acts on itself by left mult. → G embeds in S_{|G|}.

Burnside's Lemma

  • If G acts on X: # orbits = (1/|G|) · Σ_{g∈G} |X^g|
  • X^g = set of elements of X fixed by g
  • Used to count distinct necklaces/colorings up to symmetry
  • Necklace example: G = rotations (Cₙ), X = all colourings of n beads with B colours, X^g = colourings fixed by rotation g
Actions unify everything: Conjugation on elements → class equation. Conjugation on Sylow subgroups → Sylow 2 & 3. Left multiplication on cosets → Cayley's theorem. Every major result is secretly an orbit-stabilizer argument.

Semidirect Products

DEFINITION · Semidirect Product H ⋊ K

G = H ⋊_φ K is a group built from subgroups H and K where:

• G = HK, H ∩ K = {e}, H ◁ G (H normal), K not necessarily normal.

• The operation twists by φ: K → Aut(H): (h₁,k₁)(h₂,k₂) = (h₁·φ(k₁)(h₂), k₁k₂)

Direct vs Semidirect: H × K is the special case where φ is trivial (K acts on H by identity). In H ⋊ K, the action is non-trivial — K "twists" or "rewires" H.

Visual clue: In the Cayley diagram, arrows between cosets don't connect corresponding elements — the cosets are rotated/twisted versions of each other.

Identifying semidirect structure: If H ◁ G and K ≤ G with HK = G and H ∩ K = {e}, then G ≅ H ⋊ K. The homomorphism φ: K → Aut(H) is given by φ(k)(h) = khk⁻¹. If K is also normal, it's a direct product. The key question is: does conjugation by K give a non-trivial automorphism of H?

Module 3 — Quotients, Products & Homomorphisms

Foundation for Modules 4–5

Quotient Groups

DEFINITION · Quotient Group G/H

If H ◁ G (H is normal in G), then the set of left cosets G/H = {gH : g ∈ G} forms a group under the operation (aH)(bH) = (ab)H.

Order: |G/H| = |G| / |H| (this is the index of H in G).

Visualize it as "collapsing": take the Cayley diagram of G and shrink every element of each coset into a single node. The resulting diagram is the Cayley diagram of G/H. Normal subgroups are exactly the ones where this collapsing produces a well-defined group (cosets multiply consistently).
H must be normal for G/H to be a group. If H is not normal, coset multiplication is not well-defined. Always verify normality first: gHg⁻¹ = H for all g ∈ G.

Direct Products

Key Lemmas to Know

  • |A × B| = |A| · |B|
  • A × B is abelian iff both A and B are abelian
  • Components A and B are normal subgroups of A × B
  • Cₘ × Cₙ ≅ Cₘₙ iff gcd(m,n) = 1
  • Order of (a,b) in A×B = lcm(ord(a), ord(b))

Fundamental Theorem of Finite Abelian Groups

  • Every finite abelian group is a product of cyclic groups of prime power order
  • This product is unique up to rearrangement
  • To list all abelian groups of order n: factor n, then list all partition-products
  • Example: Order 12 = 4·3 gives C₁₂, C₂×C₆, C₂×C₂×C₃

Homomorphisms — Essential Facts

MUST-MEMORIZE · Homomorphism Properties

A map φ: G → H is a homomorphism if φ(ab) = φ(a)φ(b) for all a,b ∈ G.

Always true:

1. φ(e_G) = e_H — identities map to identities.

2. φ(a⁻¹) = φ(a)⁻¹ — inverses map to inverses.

3. ker(φ) = {g ∈ G : φ(g) = e_H} is a normal subgroup of G.

4. Im(φ) = {φ(g) : g ∈ G} is a subgroup of H.

5. If G is abelian, Im(φ) is abelian. If G is cyclic, Im(φ) is cyclic.

6. Order of φ(a) divides order of a.

Kernel = normality detector: The kernel of any homomorphism is always normal. Conversely, every normal subgroup N ◁ G is the kernel of the quotient map π: G → G/N. These two facts are two sides of the same coin.

Conjugacy Classes — Full Details

CONJUGACY — Definitions & Properties to Prove

a is conjugate to b if a = xbx⁻¹ for some x ∈ G. The conjugacy class of a is [a] = {xax⁻¹ : x ∈ G}.

Must know:

1. Conjugacy is an equivalence relation → conjugacy classes partition G.

2. |[a]| = [G : C_a] — size of class = index of centralizer. (Proved via orbit-stabilizer: G acts on G by conjugation; orb(a) = [a], stab(a) = C_a.)

3. a ∈ Z(G) iff [a] = {a} (conjugacy class of size 1).

4. Normal subgroups are unions of conjugacy classes. (Key exam tool: if |N| must divide |G| and N is a union of conjugacy classes, list which classes can combine to give a valid order.)

Testing normality via conjugacy classes: List all conjugacy classes of G (with sizes). A normal subgroup N must contain complete conjugacy classes and satisfy |N| | |G|. So compute: which subsets of classes have total size dividing |G|? Those are the candidates for normal subgroups. This is exactly how the A₅ class equation 60 = 1+15+20+12+12 forces A₅ to be simple — no subset of {1,15,20,12,12} (including 1) adds to a proper divisor of 60 except {1} and {1,15,20,12,12} = 60 itself.
DEFINITION · Normalizer & Centralizer

Normalizer of H in G: N_G(H) = {g ∈ G : gHg⁻¹ = H}

This is the largest subgroup of G in which H is normal. H ◁ G iff N_G(H) = G.

Center Z(G): Z(G) = {x ∈ G : xg = gx for all g ∈ G} — elements that commute with everything.

Centralizer of element a: C_a = {x ∈ G : xa = ax}

To find the normalizer quickly: if H is not normal, the normalizer must contain H and only adds elements g where gHg⁻¹ = H. Look for symmetry in the Cayley diagram — if gH = Hg (left and right cosets agree), then g ∈ N_G(H).

Module 4 — The Isomorphism Theorems

High Exam Weight
★ FIRST ISOMORPHISM THEOREM — The Most Important Theorem in the Course

Let φ: G → H be a homomorphism. Then:

G / ker(φ) ≅ Im(φ)

More precisely: the map ψ: G/ker(φ) → Im(φ) defined by ψ(g·ker(φ)) = φ(g) is an isomorphism.

Proof Scaffold — First Isomorphism Theorem

1.Define the map: Let K = ker(φ). Define ψ: G/K → Im(φ) by ψ(gK) = φ(g).
2.Well-defined: If gK = g'K, then g'⁻¹g ∈ K, so φ(g'⁻¹g) = e, so φ(g) = φ(g'). Thus ψ(gK) = ψ(g'K). ✓
3.Homomorphism: ψ((aK)(bK)) = ψ(abK) = φ(ab) = φ(a)φ(b) = ψ(aK)ψ(bK). ✓
4.Injective: ker(ψ) = {gK : ψ(gK) = e} = {gK : φ(g) = e} = {gK : g ∈ K} = {K} = identity coset. ✓
5.Surjective: Every element φ(g) ∈ Im(φ) is hit by ψ(gK). ✓
6.Conclusion: ψ is a bijective homomorphism, hence an isomorphism. □
1st Iso Theorem Recipe — How to use it:
(1) Find a surjective homomorphism φ: G → something you recognize.
(2) Compute ker(φ).
(3) Conclude G/ker(φ) ≅ Im(φ).

Classic example: det: GL_n(ℚ) → ℚ* is surjective (det of diagonal matrix = any nonzero rational). ker(det) = SL_n(ℚ). So GL_n(ℚ)/SL_n(ℚ) ≅ ℚ*.
SECOND ISOMORPHISM THEOREM (Diamond Theorem)

Let K ◁ G and H ≤ G. Then:

• HK is a subgroup of G • H ∩ K is a normal subgroup of H • HK/K ≅ H/(H ∩ K)

Use when: You have a product of subgroups HK and need to understand its structure.

Mnemonic — "Diamond": Draw a diamond with G at top, HK below G, H and K on the sides, H∩K at the bottom. The two sides of the diamond are isomorphic quotients: HK/K ≅ H/(H∩K). The lengths of the sides of the diamond are equal.
CORRESPONDENCE THEOREM (prerequisite for 3rd Iso)

If θ: G₁ → G₂ is a surjective homomorphism with K = ker(θ), then there is a bijection between:

{ subgroups of G₁ containing K } ←→ { subgroups of G₂ }

given by H ↦ θ(H) and J ↦ θ⁻¹(J). Furthermore, this bijection preserves normality and indices.

THIRD ISOMORPHISM THEOREM

If K ◁ G and K ◁ H ◁ G, then:

G/H ≅ (G/K) / (H/K)

Intuition: Quotienting in two stages (first by K, then by H/K) is the same as quotienting directly by H.

Analogy: Like (12/4)/3 = 12/12 = 1. Dividing out in stages gives the same answer.

When to Use Each Isomorphism Theorem

Theorem Trigger Phrase / Situation What You Get
1st Iso You have a homomorphism φ and want to identify a quotient group G/ker(φ) ≅ Im(φ)
2nd Iso You have H and K (one normal), and want to understand HK or count elements HK/K ≅ H/(H∩K)
3rd Iso You have a chain of normal subgroups K ◁ H ◁ G (G/K)/(H/K) ≅ G/H
Correspondence You want to transfer subgroup structure through a surjection Bijection between subgroup lattices
Exam trap: The 3rd Isomorphism Theorem requires K ◁ H AND K ◁ G AND H ◁ G — all three normality conditions. Don't just check one. Also: the conclusion is G/H ≅ (G/K)/(H/K), not "Normal A" (a noted correction from lecture).

Automorphisms

DEFINITION & KEY FACTS — Aut(G)

An automorphism is an isomorphism φ: G → G. Aut(G) is the set of all automorphisms under composition.

• Aut(G) is itself a group.

• Aut(Cₙ) ≅ U(n) (the group of units mod n), which has order φ(n) (Euler's totient).

• For prime p: Aut(Cₚ) ≅ Cₚ₋₁.

• An automorphism is determined entirely by where generators go; a generator must map to a generator of the same order.

Module 5 — Class Equation & Cauchy's Theorem

Bridge to Sylow
THE CLASS EQUATION
|G| = |Z(G)| + Σ [G : C_a]

where the sum is over one representative a from each conjugacy class NOT in Z(G), and C_a is the centralizer of a.

Note: [G : C_a] = |orb(a)| — the size of a's conjugacy class. Elements in Z(G) each form their own class of size 1.

Class equation in action: Every term on the right divides |G| (by Lagrange). This forces constraints on what p-groups can look like. Key use: if p | |G| and p | [G:C_a] for all non-central a, then p | |Z(G)|, so Z(G) is non-trivial!
THEOREM — Centre of a p-Group is Non-Trivial

If |G| = pᵏ for a prime p and k ≥ 1, then Z(G) ≠ {e}.

Proof sketch: Use the class equation. Every [G:C_a] divides |G| = pᵏ, so each term in the sum is a power of p. Since p | |G| and p | each summand, we get p | |Z(G)|, so |Z(G)| ≥ p > 1. □

THEOREM — Groups of Order p² are Abelian

If |G| = p² for a prime p, then G is abelian. In fact, G ≅ Cₚ² or G ≅ Cₚ × Cₚ.

Proof sketch: Since Z(G) ≠ {e} and |Z(G)| divides p², we get |Z(G)| = p or p². If |Z(G)| = p², done (G is abelian). If |Z(G)| = p, then |G/Z(G)| = p, so G/Z(G) is cyclic, but then G must be abelian (classic lemma) — contradiction. So Z(G) = G. □

Key lemma to memorize: "If G/Z(G) is cyclic, then G is abelian." This is used constantly. Proof: if G/Z(G) = ⟨gZ(G)⟩, then every element of G has the form gⁿz for some n and z ∈ Z(G), and any two such elements commute.
CAUCHY'S THEOREM

If a prime p divides |G|, then G has an element of order p (and hence a subgroup ≅ Cₚ).

This is the "partial converse" to Lagrange — for primes, it always works.

Counter-example to full converse: A₄ has order 12, but has no subgroup of order 6.

The Sylow Theorems

★ Most Critical for Exam

Named after Norwegian mathematician Peter Sylow (pronounced "See-Low"). These theorems give a powerful partial converse to Lagrange and allow us to classify groups by their prime-power subgroup structure.

FIRST SYLOW THEOREM — Existence

Let |G| = pᵏ · m where p is prime and p ∤ m. Then G has at least one subgroup of order pⁱ for every 1 ≤ i ≤ k.

In particular, G has at least one subgroup of order pᵏ — a Sylow p-subgroup.

SECOND SYLOW THEOREM — Conjugacy

Any two Sylow p-subgroups of G are conjugate to each other.

That is, if P and P' are both Sylow p-subgroups, then P' = gPg⁻¹ for some g ∈ G.

Key consequence: If there is only ONE Sylow p-subgroup, it must be conjugate only to itself, so it is normal in G.

THIRD SYLOW THEOREM — Count

Let nₚ = number of Sylow p-subgroups. Then:

1. nₚ ≡ 1 (mod p) 2. nₚ | m (where |G| = pᵏm, p ∤ m) 3. nₚ = [G : N_G(P)]

Corollary: If nₚ = 1, the unique Sylow p-subgroup is normal.

THE SYLOW TOOLKIT — 3-step process for any problem:
Step 1 — Factor the order: Write |G| = p₁^a₁ · p₂^a₂ · … Find all Sylow subgroup sizes.
Step 2 — Apply the 3rd theorem: For each prime p, list the possibilities for nₚ using conditions (1) nₚ ≡ 1 mod p AND (2) nₚ | (|G|/pᵏ).
Step 3 — Force nₚ = 1: If you can rule out all other values, the Sylow subgroup is normal. Then use it.

Master Reference: nₚ Analysis

Proof Strategies for Sylow Problems

Strategy: Showing a Group Has No Simple Groups of Order n

1.Factor n. Find all Sylow subgroup sizes pᵏ.
2.For each prime p, compute what nₚ could be. Check if nₚ = 1 is forced.
3.If some nₚ = 1: that Sylow subgroup is normal ⇒ G is not simple.
4.If no nₚ = 1 immediately: count elements. If total count of elements in all Sylow subgroups exceeds |G|, contradiction.
5.Use element counting: a Sylow p-subgroup of order pᵏ contains (pᵏ − 1) non-identity elements. If subgroups are distinct, their non-identity elements don't overlap (for groups of prime order). Add them up.
WORKED EXAMPLE — No Simple Group of Order 56

|G| = 56 = 2³ · 7.

Sylow 7-subgroups: n₇ ≡ 1 mod 7 and n₇ | 8. So n₇ ∈ {1, 8}.

Sylow 2-subgroups: n₂ ≡ 1 mod 2 and n₂ | 7. So n₂ ∈ {1, 7}.

Case n₇ = 1: Unique Sylow 7-subgroup is normal → G not simple. Done.

Case n₇ = 8: 8 Sylow 7-subgroups, each of order 7, any two intersecting only at e. That's 8 × 6 = 48 elements of order 7.

Remaining elements: 56 − 48 = 8. These 8 must all belong to a Sylow 2-subgroup of order 8 → n₂ = 1 → unique Sylow 2-subgroup is normal → G not simple. □

WORKED EXAMPLE — Group of Order pq (p < q, q ≢ 1 mod p)

nq ≡ 1 mod q and nq | p. Since p < q, the only value ≡ 1 mod q that divides p is 1. So nq = 1 → unique Sylow q-subgroup K ◁ G.

By similar argument (or element counting), nₚ = 1 → unique Sylow p-subgroup H ◁ G.

|H| = p, |K| = q, H∩K = {e} (coprime orders), HK = G (by order count).

Since both H and K are normal with trivial intersection: G = H × K ≅ Cₚ × Cq ≅ Cₚq. □

The "Normal + Trivial Intersection" lemma: If H ◁ G and K ◁ G and H∩K = {e} and HK = G, then G ≅ H × K. This is the bridge from "these Sylow subgroups are normal" to the explicit structure of G. You must prove hk = kh for all h∈H, k∈K — do this by showing hkh⁻¹k⁻¹ ∈ H∩K = {e}.

Orbit-Stabilizer Theorem (Crucial for Sylow Proofs)

ORBIT-STABILIZER THEOREM
|G| = |orb(x)| · |stab(x)|

Equivalently: |orb(x)| = [G : stab(x)]

The normalizer is the stabilizer: When G acts on its Sylow p-subgroups by conjugation, the stabilizer of P is N_G(P). So nₚ = [G : N_G(P)], which divides |G|.

Burnside's Lemma (for counting problems like necklaces): # of distinct orbits = (1/|G|) · Σ_{g∈G} |X^g| where X^g = fixed points of g's action on set X.

Classification of Groups

Big Picture Connections

Jordan–Hölder Theorem

  • Every finite group is "built from" finite simple groups
  • Simple groups are the building blocks of all groups
  • The classification of all finite simple groups was completed ~2004 (10,000s of pages)

Simple Groups

  • G is simple if its only normal subgroups are {e} and G
  • Cyclic groups of prime order are simple
  • A₅ (rotations of icosahedron) is simple — this is what makes the quintic unsolvable
  • A₅ has class equation: 60 = 1+15+20+12+12

Semidirect Products

  • G = H ⋊ K (not necessarily direct — arrows don't go between corresponding elements)
  • Direct product: arrows connect corresponding cosets
  • Quotient G/H "reveals" the product structure
  • Uses Aut(K) — how K "acts on" H via conjugation

Proof Construction Guide

How to Write & Evaluate Proofs

The Anatomy of a Good Proof

Proof Structure Template

①State your assumptions clearly. "Let G be a group with... Suppose H is a subgroup such that..."
②State what you will show. "We wish to show that... We claim that..."
③Build the deductive bridge. Each line must follow from what came before. Cite the result/definition you are using.
④Don't assume the conclusion. Check: "Am I using what I'm trying to prove?" (Begging the question — the most common error.)
⑤Close the proof. Explicitly state "Therefore, [exactly what was asked]." Write □ or QED.

Proof Types & When to Use Them

Proof Type When to Use Key Move
Direct proof Most algebraic proofs (homomorphism properties, subgroup checks) Assume hypotheses, manipulate, derive conclusion
Proof by contradiction Showing something is impossible (e.g., no simple group of order n) Assume ¬conclusion, derive contradiction with known facts
Induction Results about all groups of order ≤ n (Cauchy's theorem proof) Base case + inductive step on |G|
Counting / pigeonhole Element counting in Sylow arguments Count elements two ways, force contradiction or uniqueness
Construct an isomorphism "Show G ≅ H" Define map explicitly, verify homomorphism, bijective

Proving Subgroup, Normality, Isomorphism — Checklists

Subgroup Test (3 things)

  • 1. Non-empty: e ∈ H (or show at least one element)
  • 2. Closure: a,b ∈ H ⇒ ab ∈ H
  • 3. Inverses: a ∈ H ⇒ a⁻¹ ∈ H
  • Shortcut: a,b ∈ H ⇒ ab⁻¹ ∈ H (combines all three)

Normal Subgroup (pick one)

  • gHg⁻¹ = H for all g ∈ G
  • Left cosets = right cosets: gH = Hg
  • H is the kernel of some homomorphism
  • H is a union of conjugacy classes
  • [G:H] = 2 (index 2 → always normal)

Isomorphism Proof (4 things)

  • 1. Define φ: G → H explicitly
  • 2. Show φ is a homomorphism: φ(ab) = φ(a)φ(b)
  • 3. Show φ is injective: ker(φ) = {e}
  • 4. Show φ is surjective: Im(φ) = H
  • Or use 1st Iso Thm for steps 3+4

Homomorphism Proof

  • Only need: φ(ab) = φ(a)φ(b)
  • Identities and inverses are automatic
  • If defined on generators: extend linearly and check relations are preserved
  • Verify: does φ respect all group relations?

Reading & Evaluating Proofs (Exam Section)

Common Logical Errors to Spot

  • Begging the question: assumes the conclusion in the middle
  • Wrong direction: proves P→Q when Q→P was needed (e.g., injectivity vs surjectivity)
  • Local vs global: each line true, but overall structure broken
  • Syntactic ritual: correct algebra but missing logical glue
  • Counterexample as exception: one counterexample kills a universal claim

Proof Comprehension Framework

  • (1–4) Understand meaning of all terms
  • (5–6) Identify proof type & sentence roles
  • (7–8) Justify claims; find implicit assumptions
  • (9–10) Summarize overall + sub-proofs
  • (11–13) Identify modules & their logical relations
  • (14–16) Transfer method to new context
Prof's top-2 proof requirements:
1. Is it correct? (Every claim is justified, no gaps, no circular reasoning.)
2. Is it readable? (Easy to follow, correct conventions, appropriate length, no extra junk.)

Find the Error — Proof Evaluation Practice

Directly Tested on Exam

The exam includes a section where you evaluate broken proofs. Below are the exact error types from your course — practise spotting them before you see them on the paper.

Lagrange's Theorem — Broken Proofs

Broken Proof A — "Modding out" hand-wave:
"We know that when we form cosets we are modding out by H. This process naturally creates a partition, just like integers modulo n. Therefore cosets partition G and |H| divides |G|."

Error: Proof by analogy is not a proof. The argument assumes cosets behave like congruence classes without proving it. You must prove (1) every element is in at least one coset, and (2) any two cosets are equal or disjoint. The analogy is suggestive but not a deductive bridge.
Broken Proof B — Powers of h:
"Since H is a subgroup, for any h ∈ H, hⁿ ∈ G for some n. Thus |G| = n|H|."

Error: The statement "hⁿ ∈ G for some n" is trivially true (every element is in G), but says nothing useful. The author is confusing the order of an element with the index. This proves nothing about divisibility.
Broken Proof C — Cosets disjoint or equal (almost right!):
"Suppose aH ∩ bH is non-empty and aH ≠ bH. Then x = ah₁ = bh₂, so a = bh₂h₁⁻¹ ∈ bH. Therefore aH ⊆ bH. Similarly bH ⊆ aH. So aH = bH — contradiction."

Error: The algebra is actually correct, but the conclusion "this is a contradiction" is then used to say cosets must be disjoint. The phrasing "any two cosets are either disjoint nor equal" is garbled. The logical structure should say: "Two cosets are either disjoint or equal" — the proof correctly establishes this, but then mislabels the conclusion. Watch for: Local vs. Global error — each line is fine but the overall claim is stated wrong.

Prime Order Groups — Broken Proofs

Broken Proof D — "Limited structure forces uniqueness":
"By Lagrange, any subgroup has order 1 or p. So G has no non-trivial subgroups. This limited structure forces G to be unique."

Error: The conclusion ("forces G to be unique") does not follow. Having no non-trivial subgroups tells you nothing directly about cyclicity or isomorphism class. The correct argument: pick any a ≠ e in G. Then ⟨a⟩ is a subgroup of order dividing p. Since p is prime and a ≠ e, ord(a) = p. So ⟨a⟩ = G, meaning G is cyclic.
Broken Proof E — "Same number of elements = isomorphic":
"Let G and H both have order p. Since any bijection between sets of the same size preserves the number of elements, G and H are isomorphic."

Error: A bijection of sets does NOT imply a group isomorphism. Isomorphism requires the bijection to preserve the group operation: φ(ab) = φ(a)φ(b). This proof confuses set-bijection with group-isomorphism — a classic syntactic ritual error.
The correct proof that all groups of prime order p are isomorphic to Cₚ:
Let G have order p (prime). Pick any a ∈ G with a ≠ e. By Lagrange, ord(a) | p. Since p is prime, ord(a) = 1 or p. Since a ≠ e, ord(a) ≠ 1, so ord(a) = p. Then ⟨a⟩ is a subgroup of order p = |G|, so ⟨a⟩ = G. Thus G is cyclic of order p, and all such groups are isomorphic to Cₚ. □

Connections — Fermat's Theorem & RSA

Module 4 Week 1 / Week 9

These connections show the power of abstraction — one group-theoretic fact gives you a family of classical number theory theorems for free.

FERMAT'S LITTLE THEOREM — via Group Theory

Group-theoretic statement: If G is a group of order n, then a^n = e for every a ∈ G.

Apply to G = U(p) = ℤₚ* (units mod p, p prime):

|U(p)| = p−1. So for any a with gcd(a,p) = 1: a^{p-1} ≡ 1 (mod p).

This is Fermat's Little Theorem — a direct corollary of Lagrange!

U(pq) ≅ U(p) × U(q) — Euler's Totient via Isomorphism

The canonical map f: U(pq) → U(p) × U(q) sending ā ↦ (ā mod p, ā mod q) is a group isomorphism (for distinct primes p,q).

Consequence: φ(pq) = |U(pq)| = |U(p)| · |U(q)| = (p−1)(q−1)

The isomorphism is proved via: (1) define f, (2) show homomorphism, (3) show injective (kernel = {1}), (4) show surjective by CRT (Chinese Remainder Theorem gives the inverse).

RSA connection (exam may ask about this): RSA encryption uses the fact that if n = pq and gcd(a,n) = 1, then a^{φ(n)} ≡ 1 (mod n). This follows from a^|G| = e applied to G = U(n). Decryption works because ed ≡ 1 (mod φ(n)), so c^d = (m^e)^d = m^{ed} = m^{1+kφ(n)} = m · (m^{φ(n)})^k ≡ m (mod n).
